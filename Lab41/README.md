Данная лабораторная работа в себе обединяет работы 4 и 5. В 4 был создан графический интерфейс для программы с лабораторной работы 3, а в 5 была приделана база данных и программа для управления ею.

Данная программа является графическим приложением для учёта товаров написанной на языке java. К ней подключена база данных sqlite, в ней хранятся данные о товарах, которые были добавлены.
Графика написана на java.swing

Кнопка SCAN нужна для будущего порта на другую платформу
